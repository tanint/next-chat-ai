## Vercel AI SDK 
ตัวช่วยทำ UI สำหรับทำ chat UI แบบที่ chatpgt ทำได้ง่ายๆ
เราสามารถแสดง Response ได้ 2 แบบ
- Streaming (Text ไหลๆ มาแบบ ChatGPT) 
- หรือแบบ Blocking แบบรอให้เสร็จแล้วโชว์ทีเดียว
